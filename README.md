# Media Performance Training — Slide Deck

Bộ slide thuyết trình nội bộ về **Media Performance** cho team Account.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Mở http://localhost:5173/

## 🎯 Nội dung (32 slides)

| Module | Slides | Nội dung |
|--------|--------|----------|
| Opening | 01-03 | Title, Agenda, Quote |
| KPI Fundamentals | 04-10 | Definitions, Funnel, Awareness/Engagement/Conversion KPIs |
| Platforms & Formats | 11-22 | 8 nền tảng chi tiết (Meta, Google Search, GDN, YouTube, TikTok, Zalo, Display, LinkedIn) |
| Optimization | 23-31 | Creative, Targeting, Budget, Content Guide, Mistakes, Checklist |
| Closing | 32 | Thank you & Q&A |

## 🖥️ Navigation

- **→ / Space / Enter** = Next slide
- **← / ↑** = Previous slide
- **Click** phải/trái = Next/Previous
- **Tab/Button click** = Tương tác trong slide (không chuyển slide)

## 🛠 Tech Stack

React 19 • Vite 6 • TypeScript • TailwindCSS v4 • Framer Motion

## 📐 Design

- **Dark theme** (`#1a1a1a`) + Orange accent (`#FF5722`)
- **Fonts:** Montserrat (display) + Be Vietnam Pro (body) — hỗ trợ tiếng Việt đầy đủ
- **Slide size:** 1280×720px, tự co giãn theo màn hình

## 📦 Build

```bash
npm run build
```

Output: `dist/` — deploy lên Vercel/Netlify.
